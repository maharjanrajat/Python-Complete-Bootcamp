## Vscode Shortcuts

| Shortcut | Description |

| `Ctrl + Shift + E` | Toggle File Explorer |

| `Ctrl + B` | Toggle Side Bar |
| `Ctrl + 1` | Focus on editor |
| `Ctrl + ~` | Toggle Terminal |
